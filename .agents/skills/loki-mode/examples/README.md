# Loki Mode Demo Apps

These apps were generated by Loki Mode from a single prompt. Each is a self-contained HTML file with embedded CSS and JavaScript -- no external dependencies, no build step.

## Demo Apps

| Demo | Description | File |
|------|-------------|------|
| SaaS Dashboard | Analytics dashboard with KPI cards, sparklines, revenue chart, activity feed | `demo-saas-dashboard.html` |
| Task Manager | Kanban board with drag-and-drop, localStorage persistence, add/delete tasks | `demo-task-manager.html` |
| Chat App | Real-time chat interface with typing indicators, auto-scroll, message bubbles | `demo-chat-app.html` |
| Landing Page | Product landing page with hero, features, pricing, FAQ accordion, newsletter | `demo-landing-page.html` |
| Portfolio | Developer portfolio with animated gradients, skill bars, project cards, contact form | `demo-portfolio.html` |
| E-Commerce | Product listing with filters, sorting, star ratings, cart counter | `demo-ecommerce.html` |

## How to View

Open any HTML file directly in your browser:

```bash
# macOS
open examples/demo-saas-dashboard.html

# Linux
xdg-open examples/demo-saas-dashboard.html

# Windows
start examples/demo-saas-dashboard.html

# Or use a local server
cd examples && python3 -m http.server 8080
```

## Features

Every demo includes:

- **Self-contained** -- Single HTML file, no external dependencies
- **Dark mode** -- Toggle button with OS preference auto-detection
- **Responsive** -- Works on mobile, tablet, and desktop
- **View Source** -- Built-in button to inspect the source code
- **Professional UI** -- Clean typography, consistent spacing, polished design
- **No build step** -- Open in any browser and it just works

## Use as Starter Templates

These demos serve as starting points for real projects. Copy any file and customize it:

1. Pick a demo closest to your target app
2. Copy the HTML file to your project
3. Modify the content, colors, and functionality
4. Split into separate CSS/JS files as the project grows
